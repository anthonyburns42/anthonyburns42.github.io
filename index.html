Code I'm proud of

Air Force Reserve Command

Robins AFB, GA • 2018 - Current

In late 2020, our leadership started a command-wide process improvement initiative that requires metrics from each department and division. While my primary duties are that of graphic design and illustration, I knew this would be the perfect opportunity to use some of my coding knowledge to create a vanilla JavaScript application (for my own use) to assist me in compiling and reporting the metrics for our division. I receive data in the form of Excel sheets from multiple designers cataloging their projects and customer survey responses which I save as CSV files. My custom JavaScript application parses, compiles and graphs the monthly CSV data which I then return to my supervisor as png screenshots of the graphs.

CypherWorx

Rochester, NY • 2016 - 2018

I was fortunate to be able to create the Learning Paths feature for the CollaborNation Learning Management System (LMS) from start to finish. I worked on Learning Paths for over a year. I created all the MySQL databases, the back-end PHP and Drupal code, and the front-end HTML, CSS, jQuery for all the various Learning Paths pages within the LMS. Learning Paths allows administrators to group, edit, and order a series of courses, events and resources for user completion. At the end of a Learning Path, the user receives a certificate and all credit for the completed courses. I developed six web pages in total for Learning Paths; LP Catalog, Create LP, Edit LP, Create LP Step, Edit LP Step, and LP Details/Reports. It was the most challenging thing I have done in coding so far and I am super proud to say it is still a feature of CollaborNation today!
